Project's Information

Initiative

The initiative for the creation of this project was to build an example of a static website.

Introduction & Goals

This project is a static photography portfolio website built to showcase my hobby photography to the public. It is also a concept project for showcasing my knowledge of the following technologies HTML5, CSS3, JavaScript, and Bootstrap.

Project's Features

Features

Project should contain the following features:

  1. Photo gallery.

  2. Lazy loading of images.

  3. Ability to view images in full size.

  4. Ability to filter images by category.
